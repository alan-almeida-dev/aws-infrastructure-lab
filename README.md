# AWS Infrastructure Lab

Projeto prático realizado na AWS com o objetivo de compreender os principais componentes de rede e computação em ambientes Cloud.

## Recursos Criados

- VPC
- Subnet Pública
- Internet Gateway
- Route Table
- Security Group
- EC2 Amazon Linux 2023
- Acesso SSH

## Principais Aprendizados

- Criação e configuração de VPC
- Segmentação de rede com Subnets
- Configuração de Internet Gateway
- Gerenciamento de rotas
- Segurança utilizando Security Groups
- Acesso remoto via SSH
- Troubleshooting de conectividade e autenticação
